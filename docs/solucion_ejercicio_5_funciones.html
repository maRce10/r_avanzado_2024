1 5.1 Cree una función que calcule el promedio, la desviación estándar y el error estándar de un vector numérico. Estos valores deben ser devueltos como una lista con nombres.

Código 
ee_fun <- function(x){
  sd(x) / sqrt(length(x))
}

f1 <- function(x) {
  
  prom <- mean(x)
  de <- sd(x)
  ee <- ee_fun(x)
  
  resultado <- list(prom = prom, de = de, ee = ee)

  return(resultado)
  }

f5 <- function(x, na.rm = FALSE, col = "tomato", suma = FALSE) {
